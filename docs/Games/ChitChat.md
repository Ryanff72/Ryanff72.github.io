# Chit Chat

## About 
Chit Chat is a conversation simulation / point and click game. Made for the [Game By It's Cover 2020 Game Jam](https://itch.io/jam/a-game-by-its-cover-2020)! By downloading, you will be granted access to our groundbreaking test facility, in which you will get to learn about how to talk to some of our most interesting employees!
