# Inputs
Inputs allow users to input data into your program. This data can be stored in the form of **variables**. Can use <code> input() </code> to get input from the user. To assign this input to a variable, we can type:<br><br>
<code> 
  variable = input()
</code><br><br>
When this code is ran, it will wait for input from the user. After the user provides an input, lets print the variable back to the user. Add the following line of code.<br><br>
<code>
  print(variable)
</code><br><br>
This will repeat what the user typed back to them.<br><br>
